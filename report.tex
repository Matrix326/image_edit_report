\begin{abstract}
图像编辑（Image Editing）任务以原始图像和自然语言编辑指令作为输入，输出修改后的图像。该任务不仅要求模型改对，还要求未被指令涉及的区域尽量保持不变。围绕这一目标，本报告首先对 InstructPix2Pix、Qwen-Image-Edit 与 EditAR 三类代表性基线进行统一复现，并在 MagicBrush 开发集样例上进行评估。针对三种基线模型的不足，我们进行如下改进：基于训练的方法，通过减少采样步数加速模型推理，对于背景退化问题，我们设计了adapter学习低步数到高步数到修正，在加速到同时保证编辑效果，我们设计进行了lora微调，通过基本的真值作为优化目标到增强mask约束到混合专家，指标和编辑效果均有提升；基于推理的方法，我们通过prompt改写加多参数候选，mask裁剪和背景融合三种策略，使推理结果专注修改区域，最后进一步进行图层分解，分层编辑来优化编辑效果。
\end{abstract}
\keywords{图像编辑；多模态大模型；扩散模型；自回归模型；推理优化；模型微调；}

\section{Introduction}

指令式图像编辑（Instruction-Guided Image Editing）要求模型在给定原始图像与自然语言编辑指令的条件下生成编辑后图像。与普通图像生成不同，该任务同时包含两个约束：一方面，模型需要准确理解并执行指令中指定的编辑意图；另一方面，未被指令涉及的背景、主体结构、光照纹理和局部细节应尽量保持不变。因此，图像编辑的核心难点并不只是“能否改动”，还包括“是否改在正确区域”以及“是否避免无关区域被重绘”。

首先在 MagicBrush~\cite{magicbrush} 数据集上统一复现三类代表性基线模型：InstructPix2Pix（简称 P2P）代表扩散式端到端指令编辑路线~\cite{instructpix2pix}；Qwen-Image-Edit 代表多模态大模型驱动的语义编辑路线~\cite{qwenimage}；EditAR 代表离散视觉 token 与自回归 Transformer 结合的统一条件生成路线~\cite{editar}。三类方法分别体现了扩散生成、多模态理解与自回归视觉建模在图像编辑任务中的不同优势与局限。

实验评估围绕目标对齐、背景保持、编辑区域变化和语义一致性展开，主要使用 SSIM、PSNR、BG-SSIM、Edit-Region Change 与 CLIP Score 等指标。根据复现结果，Qwen-Image-Edit 在 SSIM、PSNR 和 CLIP 上整体更优，说明其目标对齐和语义执行能力较强，但 BG-SSIM 较低，容易出现背景、风格或非目标区域的越界修改；InstructPix2Pix 背景保持较好，但复杂替换、小目标新增和大范围语义变化中容易编辑不足；EditAR 的保持类指标不低，但定性结果显示其更接近保守失败，即保持了输入结构却没有充分完成编辑目标。

在基线分析基础上，本文进一步整理多条改进尝试：针对 Qwen-Image-Edit 的低步数推理质量退化，采用从高步数 teacher 到低步数 student 的蒸馏修正思路；针对 Qwen2511 的局部编辑和背景保持问题，设计 GT LoRA、MTP LoRA 与 MoE-Fusion Teacher LoRA 三种微调方案；针对 InstructPix2Pix 的单次采样不稳定问题，引入 prompt 改写、多候选搜索、mask-crop 和背景融合；针对对象级局部编辑，探索图层分解与分层编辑；针对 EditAR，则分析其 VQ-VAE 重建误差与自回归误差累积带来的固有限制。

\section{Related Work}

\subsection{扩散模型与图像编辑}

\paragraph{扩散模型基础。}
扩散模型（Diffusion Models）通过定义前向加噪过程 $q(x_t|x_{t-1})$ 和反向去噪过程 $p_\theta(x_{t-1}|x_t)$ 来建模数据分布。DDPM~\cite{ddpm} 将前向过程定义为逐步添加高斯噪声的马尔可夫链，反向过程则训练一个 UNet 网络 $\epsilon_\theta(x_t,t)$ 预测所添加的噪声，通过最小化 $\mathbb{E}_{x_0,\epsilon,t}\left[\|\epsilon-\epsilon_\theta(x_t,t)\|^2\right]$ 来优化。采样时从纯噪声 $x_T\sim\mathcal{N}(0,I)$ 出发，逐步去噪得到生成图像 $x_0$。

Latent Diffusion Model (LDM)~\cite{ldm} 将扩散过程从像素空间迁移到 VAE 压缩的潜空间。其架构包含三个核心组件：(1) 一个预训练的 VAE 编码器 $E$ 将图像压缩到潜表示 $z=E(x)$；(2) 一个在潜空间运行的 UNet 去噪网络 $\epsilon_\theta(z_t,t,\tau(c))$，通过交叉注意力层接收条件 $c$（文本、图像等）；(3) 一个 VAE 解码器 $D$ 将去噪后的潜表示恢复为图像。LDM 大幅降低了高分辨率合成的计算开销，成为 Stable Diffusion 等主流文生图模型的骨干架构。

\paragraph{可控图像生成。}
ControlNet~\cite{controlnet} 在冻结的预训练扩散模型旁添加一个可训练的“零卷积”副本编码器，将边缘图、深度图、姿态骨架等空间条件注入模型，实现了对生成结构的精细控制而无需微调原始权重。IP-Adapter~\cite{ipadapter} 提出解耦的交叉注意力机制，将图像特征和文本特征分别通过独立的交叉注意力层注入 UNet，使得模型能够同时接受文本提示和图像提示（如风格参考图），在保持文本可控性的同时实现图像风格与内容迁移。

\paragraph{文本驱动的图像编辑。}
Prompt-to-Prompt（P2P）~\cite{p2p} 发现了扩散模型交叉注意力层的核心性质——注意力图编码了文本 token 与图像空间区域的对应关系。基于此，P2P 提出通过操控交叉注意力图来实现文本驱动的图像编辑：替换某个 token 的注意力图即可替换对应物体，对注意力图加权可控制编辑强度，注入新的注意力图可实现物体添加。该方法无需额外训练或 mask 标注。

SDEdit~\cite{sdedit} 通过在输入图像上加入噪声并执行文本条件反向去噪，实现基于随机微分方程的图像合成与编辑，是扩散式图像编辑中的早期代表方法之一。

\paragraph{InstructPix2Pix。}
InstructPix2Pix~\cite{instructpix2pix} 首次实现了端到端的指令式编辑。其训练流程为：(1) 使用 GPT-3 基于图像描述生成编辑指令和编辑后描述；(2) 使用 Prompt-to-Prompt 和 Stable Diffusion，以原始描述和编辑后描述分别生成编辑前后的图像对；(3) 以这些自动生成的图像对作为训练数据，训练一个以输入图像 $x$ 和编辑指令 $c$ 为双条件的扩散模型。推理时无需 DDIM 反演或 mask 标注，直接将输入图像和指令送入模型进行单次前向采样即可。本文选取其作为扩散式 baseline。

\subsection{多模态大模型与图像编辑}

随着大语言模型的成功，研究者开始将 LLM 的语义理解能力与图像生成模型结合，形成了多模态大模型驱动的图像编辑路线。

\paragraph{Qwen-Image。}
Qwen-Image~\cite{qwenimage} 通过大规模多任务训练统一了文生图、图生图和图像重建三种能力。其核心设计为双编码器架构：(1) 语义编码器负责理解文本指令与图像的高级语义内容，提供编辑所需的语义指导；(2) 重建编码器专门保留输入图像的像素级细节信息（如纹理、边缘、颜色分布），以确保编辑后非目标区域的一致性。模型推理时，双编码器分别提取语义特征和重建特征，融合后作为条件注入扩散去噪过程，在“编辑得对”和“保持不变”之间取得平衡。

\paragraph{其他多模态编辑模型。}
EMU2~\cite{emu2} 将多模态理解（图像描述、VQA）与生成（文生图、编辑）整合到单一自回归模型中，支持以对话方式进行多轮图像编辑。GILL~\cite{gill} 提出将 LLM 的文本嵌入通过一个可学习的映射网络转换到图像生成模型的输入空间，实现了 LLM 对 Stable Diffusion 的直接控制——用户用自然语言描述编辑需求，LLM 生成对应的条件嵌入，驱动扩散模型完成编辑。

本文选取 Qwen-Image-Edit 作为多模态大模型式 baseline，因其双编码器设计明确区分了语义理解与细节保持，代表性较强。

\subsection{自回归视觉生成与编辑}

与扩散模型的逐步去噪不同，自回归模型将图像生成建模为序列预测问题，形成了与扩散模型互补的另一条技术路线。

\paragraph{视觉 Token 化。}
VQGAN~\cite{vqgan} 奠定了自回归视觉生成的基础架构：首先使用 VQ-VAE 将图像编码为离散的视觉 token 序列（codebook indices），然后训练一个自回归 Transformer（如 GPT）以 next-token prediction 方式逐 token 生成图像序列，最后通过 VQ-VAE 解码器将 token 序列恢复为图像。该架构使标准 NLP 自回归范式可直接应用于视觉生成。

\paragraph{大规模自回归文生图。}
DALL-E~\cite{dalle} 首次展示了自回归模型在大规模文生图任务上的强大能力，使用 120 亿参数的 Transformer 在 2.5 亿图文对上训练。Parti~\cite{parti} 将模型规模扩展至 200 亿参数，验证了自回归文生图遵循 scaling law——模型越大，生成质量越高、细节越丰富。LlamaGen~\cite{llamagen} 进一步证明了标准 Llama 架构无需视觉特化设计即可实现与扩散模型竞争的自回归图像生成。

\paragraph{EditAR。}
EditAR~\cite{editar} 将自回归范式扩展到统一条件生成框架。其核心思路是将输入图像、条件信号（如编辑指令、深度图、边缘图）和输出图像全部表示为统一的离散 token 序列，通过单一自回归 Transformer 处理多种条件生成任务。该框架的优势在于任务形式高度统一——图像编辑、深度估计、边缘检测、分割到图像等任务共用同一个模型，无需为每种任务设计独立的扩散结构。本文选取 EditAR 作为自回归式 baseline，以评估该范式在指令式图像编辑中的实际表现。

\subsection{评测基准}

MagicBrush~\cite{magicbrush} 是首个大规模人工标注的指令式图像编辑数据集，约 10,000 条样例，每条配备输入图像、编辑指令、人工编辑的目标图像和编辑区域 mask，覆盖背景替换、物体增删、属性修改、姿态变化等类别，其多轮编辑子集支持序列化编辑场景。本文使用其开发集 60 个样例作为评测基准。

\section{Baseline Reproduction}

\subsection{评测指标与实验设置}

\subsubsection{评测指标}

设输入图像为 $I\in\mathbb{R}^{H\times W\times 3}$，模型输出为 $O$，人工目标图像为 $G$，编辑区域二值 mask 为 $M\in\{0,1\}^{H\times W}$（$M_{ij}=1$ 表示像素 $(i,j)$ 属于编辑区域）。采用以下五类指标：
\begin{itemize}
  \item \textbf{SSIM / PSNR (Target, Output)：}衡量 $O$ 与 $G$ 的结构相似度和像素级保真度，反映编辑输出与人工目标的整体一致性。SSIM~\cite{ssim} 从亮度、对比度、结构三个维度综合评估：
\[
\operatorname{SSIM}(x,y)=
\frac{(2\mu_x\mu_y+C_1)(2\sigma_{xy}+C_2)}
{(\mu_x^2+\mu_y^2+C_1)(\sigma_x^2+\sigma_y^2+C_2)}.
\]
  \item \textbf{BG-SSIM：}仅在非编辑区域（$M_{ij}=0$）计算 SSIM，衡量模型对背景的保持能力。该指标直接反映模型是否在无需修改的区域引入了不必要的变化。
  \item \textbf{SSIM / PSNR (Input, Output)：}衡量 $O$ 与 $I$ 的整体相似度——值越高说明模型对原图的改动越小（越保守）。需结合编辑指令判断：保守不一定好（可能编辑不足）。
  \item \textbf{Edit-Region Change：}编辑区域内的平均像素绝对变化量，需结合指令语义解读——过小表示编辑不足，过大可能表示过度修改。
  \item \textbf{CLIP Score (Target, Output)：}使用 CLIP~\cite{clip} 的 ViT-L/14 图像编码器提取 $O$ 与 $G$ 的图像特征，计算余弦相似度，衡量语义级别的编辑一致性。这是对像素级指标的重要补充。
\end{itemize}

\subsubsection{实验设置}

三种 baseline 均使用官方发布的预训练权重，不做任何微调。P2P 基于 Stable Diffusion v1.5 骨干，扩散步数 $T=100$，分类器无关引导强度 $\omega=7.5$，图像引导强度和文本引导强度均设为 1.5。Qwen-Image-Edit 使用 Qwen-Image 官方推理配置，输入分辨率统一缩放至 $512\times512$，采用默认的双编码器融合权重。EditAR 使用 VAR-d30 自回归模型，top-$k=100$、top-$p=0.9$，图像 token 序列长度 $N=1024$。全部实验在单卡 NVIDIA A100-80GB GPU 上完成，推理时关闭梯度计算以最大化显存利用。

\subsection{InstructPix2Pix（扩散式）}

\subsubsection{技术原理}

InstructPix2Pix~\cite{instructpix2pix} 是基于 Stable Diffusion 的条件扩散模型。其核心创新在于训练数据的自动构造和双条件注入机制。

\paragraph{训练数据构造流水线。}
训练过程无需任何人工标注：
\begin{itemize}
  \item 步骤 1：使用 BLIP 等模型为真实图像生成文本描述；
  \item 步骤 2：使用 GPT-3 基于描述生成编辑指令 $c$ 和编辑后描述 $t_{\mathrm{edit}}$；
  \item 步骤 3：使用 Prompt-to-Prompt 技术，以原始描述 $t_{\mathrm{orig}}$ 和 $t_{\mathrm{edit}}$ 分别通过 Stable Diffusion 生成编辑前后的图像对 $(x_{\mathrm{before}},x_{\mathrm{after}})$。
\end{itemize}

\paragraph{模型架构。}
模型以输入图像 $x$ 和编辑指令 $c$ 为双条件。输入图像通过 VAE 编码器得到潜表示，与噪声拼接后送入 UNet。编辑指令经 CLIP 文本编码器编码后通过交叉注意力注入 UNet。模型学习预测添加到潜表示上的噪声，去噪后经 VAE 解码器得到编辑后图像 $y$。形式化地：
\[
p_\theta(y\mid x,c)=
\int p_\theta(z_T)\prod_{t=1}^{T}
p_\theta(z_{t-1}\mid z_t,E(x),\tau(c))\,dz_{1:T},
\qquad y=D(z_0).
\]
其中 $E$ 为 VAE 编码器，$D$ 为解码器，$\tau$ 为文本编码器。推理时采用分类器无关引导（CFG），同时以空图像和空文本为负条件，通过两个引导尺度分别控制图像条件强度和文本条件强度。

\subsubsection{典型案例}

P2P 处于 Qwen 与 EditAR 之间的中间态：不如 Qwen 精准，但比 EditAR 的语义理解强得多。在简单属性编辑上表现极佳（dev\_05 SSIM 0.9919），且背景保持为三种方法最优（BG-SSIM 0.7952）。但在复杂空间推理（dev\_31 “bird on zebra head”，SSIM 0.412）和局部替换（dev\_37 “mirror$\rightarrow$window”，SSIM 0.107，全数据集 P2P 最差）中容易全局失控。60 样例平均 SSIM 0.7228，处于不上不下的位置。

\subsection{Qwen-Image-Edit（多模态大模型式）}

\subsubsection{技术原理}

Qwen-Image-Edit 基于 Qwen-Image~\cite{qwenimage} 多模态基础模型。Qwen-Image 通过大规模多任务训练（文生图、图生图、图像重建）在统一框架内同时优化生成质量与编辑一致性。

\paragraph{双编码器架构。}
Qwen-Image 的核心创新在于解耦了语义理解和细节保持：
\begin{itemize}
  \item \textbf{语义编码器：}基于 Qwen-VL 视觉语言模型，负责理解输入图像的高级语义内容（物体类别、场景布局、人物身份等）以及解析编辑指令的意图。它将图像和文本指令映射到共享的语义特征空间。
  \item \textbf{重建编码器：}专注于保留输入图像的像素级细节——纹理、边缘、颜色分布、光照条件等低层视觉特征。该编码器确保模型在语义层面进行编辑时不会丢失输入图像的视觉一致性。
\end{itemize}

\paragraph{推理流程。}
给定输入图像 $I$ 和编辑指令 $T$：(1) 语义编码器从 $I$ 和 $T$ 中提取语义特征 $f_{\mathrm{sem}}$；(2) 重建编码器从 $I$ 中提取重建特征 $f_{\mathrm{rec}}$（保持细节信息）；(3) $f_{\mathrm{sem}}$ 和 $f_{\mathrm{rec}}$ 通过可学习的融合权重拼接后，作为条件注入扩散模型的 UNet 交叉注意力层；(4) 扩散过程在条件约束下生成编辑后图像。

\paragraph{关键特性。}
双编码器带来的优势是：当编辑指令语义明确时，语义编码器主导编辑方向；当编辑指令不涉及某些区域时，重建编码器自动保持这些区域的像素一致性。但缺点在于：当重建编码器权重过大时，模型倾向于“过度保守”——即使在需要修改的区域也尽量保持原样，导致编辑幅度不足；而当语义编码器主导时，也可能引发全局性的风格重绘——模型不仅修改了指定区域，还将整个图像的色调、光照、纹理都向训练数据中见过的风格偏移。

\subsection{EditAR（自回归式）}

\subsubsection{技术原理}

EditAR~\cite{editar} 采用自回归范式进行图像编辑，与扩散模型从噪声逐步去噪不同，自回归模型将图像视为离散 token 序列，以 next-token prediction 方式逐 token 生成。

\paragraph{统一 Token 化。}
EditAR 的核心设计是将所有模态统一表示为离散 token：
\begin{itemize}
  \item \textbf{图像 Token 化：}使用预训练的 VQ-VAE 将输入图像 $I$ 编码为 $N$ 个离散 token $\{v_1,v_2,\ldots,v_N\}$，每个 token 来自大小为 $K$ 的 codebook。同样将目标输出图像表示为 token 序列。
  \item \textbf{条件 Token 化：}编辑指令 $c$ 经文本 tokenizer 编码为文本 token 序列。对于非文本条件（如深度图、边缘图），同样使用对应的 VQ-VAE 编码为 token。
\end{itemize}

\paragraph{自回归生成。}
所有 token 拼接为一个统一的序列，送入自回归 Transformer 进行 next-token prediction 训练。Transformer 学习条件分布：
\[
p(\{y_i\}_{i=1}^{N}\mid I,c)
=
\prod_{i=1}^{N}p(y_i\mid y_{<i},\operatorname{tok}(I),\operatorname{tok}(c)).
\]
推理时，模型从条件 token（输入图像 + 编辑指令）开始，逐 token 自回归生成输出图像的 token 序列，最后通过 VQ-VAE 解码器将 token 恢复为图像。

\paragraph{统一条件生成。}
EditAR 的关键优势在于单一框架处理多种任务：只需改变条件类型（编辑指令、深度图、边缘图、分割图），同一模型即可完成图像编辑、深度估计、边缘检测到图像、分割到图像等不同任务，无需为每种任务设计专门的架构或训练流程。

\section{Method Improvement}

\subsection{Qwen-Image-Edit 低步数蒸馏加速}
\label{sec:distill}

Qwen-Image-Edit 在高步数推理下通常具有较好的编辑质量，但推理耗时较长；当采样步数降低到 4-step 时，虽然速度明显提升，但容易出现背景退化、纹理漂移和局部细节损失。我们的实验观察表明，4-step 的主要失败并非语义理解完全错误，而是低步数去噪路径带来的局部质量退化。因此，本节使用 40-step 输出作为 teacher，4-step 输出作为 student，通过轻量后处理模块学习从低步数结果到高步数结果的修正，在不修改 Qwen 主干模型的前提下降低推理成本。

\subsubsection{动机}

在 MagicBrush-dev 60 个样例上，Qwen-Image-Edit 40-step 具有较高编辑精度，但单样本推理耗时约 130 秒；直接降到 4-step 后，单样本耗时约 13.7 秒，但 SSIM、PSNR 和 BG-SSIM 明显下降。由于 4-step 与 40-step 输出来自同一模型，它们共享相同的语义理解能力和生成先验，差异主要集中在高频细节、边缘锐度和局部背景一致性。因此，我们将问题建模为图像空间的自蒸馏修正：冻结 Qwen，仅训练一个轻量 adapter，将快速 student 输出精炼到接近 teacher 的质量。

\paragraph{方法框架。}
设原图为 $I$，编辑指令为 $p$，4-step student 输出为 $S=Qwen_4(I,p)$，40-step teacher 输出为 $T=Qwen_{40}(I,p)$。低步数蒸馏模块记为 $A_\theta$，目标是从 $(I,S)$ 中预测接近 teacher 的结果：
\[
\hat T=A_\theta(I,S),\qquad
\mathcal{L}=\mathcal{L}(\hat T,T).
\]
为了使修正尽量局部且可控，adapter 采用 \textbf{gated residual} 形式，输入为原图、student 输出和二者差分的拼接：
\[
X=\operatorname{concat}(I,S,S-I)\in\mathbb{R}^{H\times W\times 9}.
\]
模型输出修正量 $\delta$ 与空间门控 $\alpha$，最终结果为：
\[
\hat T=S+\alpha\odot\delta.
\]
其中 $\delta=\tanh(\delta_{\mathrm{raw}})\times0.25$ 用于限制单像素修正幅度，$\alpha=\sigma(\alpha_{\mathrm{raw}})$ 用于控制哪些区域需要修正。三路输入的作用各不相同：$I$ 提供原图结构参考，$S$ 提供低步数编辑结果，$S-I$ 显式提示模型已经发生变化的位置。残差门控使 adapter 更倾向于“少改但有效”，避免退化为对整幅图像的重绘网络。

\paragraph{训练目标。}
训练损失由像素重建、边缘纹理、结构一致性和门控稀疏项组成：
\[
\mathcal{L}
=\mathcal{L}_{\mathrm{recon}}
+0.25\mathcal{L}_{\mathrm{edge}}
+0.05\mathcal{L}_{\mathrm{ssim}}
+0.02\mathcal{L}_{\alpha}.
\]
其中 $\mathcal{L}_{\mathrm{recon}}$ 约束输出接近 teacher，$\mathcal{L}_{\mathrm{edge}}$ 强化边缘和纹理，$\mathcal{L}_{\mathrm{ssim}}$ 约束整体结构，$\mathcal{L}_{\alpha}$ 鼓励只在必要区域启用修正。该方法的定位是用极小额外开销弥补低步数推理的质量损失。

\paragraph{训练配置。}
训练数据来自 MagicBrush~\cite{magicbrush} 训练集。训练前预先缓存每个样本的 Qwen-4step student 输出与 Qwen-40step teacher 输出，二者均使用 $\mathrm{cfg}=4.0$。Adapter 使用 AdamW 优化器，学习率 $3\times10^{-5}$，batch size 8；base4-to-base40 分支训练 54k+ optimization steps，open\_lora4-to-base40 分支配置保持一致。由于 Qwen 主干完全冻结，训练成本主要来自轻量卷积 adapter，远低于完整扩散模型微调。

\subsubsection{小结}

本节补充的低步数蒸馏实验说明，Qwen-Image-Edit 的推理加速可以从两个层面实现：一是通过轻量 Gated Residual Adapter 在图像空间修复 4-step 输出，带来约 $9.4\times$ 加速且显著提升背景保持；二是通过开源 Distill-LoRA 直接改善低步数去噪轨迹，将总耗时进一步压缩到约 6.8 秒。前者部署成本低、适合作为通用后处理模块；后者质量上限更高，适合作为最终高性能推理配置。


\subsection{Qwen2511 GT/MTP/MoE 微调方案}

本节围绕更新的 Qwen 系列模型 Qwen-Image-Edit-2511 在 MagicBrush 指令图像编辑任务上的适配与改进展开。基线复现表明，Qwen 系列模型具有较强的语义理解和编辑能力，但在 MagicBrush 这类强调局部编辑与背景保持的任务中，仍容易出现编辑区域定位不准、非目标区域被重绘等问题。为此，本节开展三条路线研究：首先使用 GT LoRA 验证参数高效微调的有效性；随后通过 MTP 构造更适合局部编辑的监督目标；最后通过 MoE-Fusion Teacher 将多个专家候选的区域优势离线蒸馏到单一 Qwen LoRA 模型中。

\subsubsection{实验设置、统一接口与指标总览}

MagicBrush 中每个样本可表示为 $(I,p,G,M)$，其中 $I$ 为原图，$p$ 为编辑指令，$G$ 为人工目标图像，$M\in\{0,1\}^{H\times W}$ 为编辑区域 mask。实验使用 MagicBrush train 的 8807 条样本进行训练，使用 dev 的 528 条样本进行测试；训练和推理均基于 Qwen-Image-Edit-2511 图像编辑接口，最大像素数设为 $512^2$。

训练数据在 DiffSynth/Qwen-Image-Edit 的 metadata 中统一写成：
\[
\texttt{edit\_image}=[I],\quad
\texttt{prompt}=p,\quad
\texttt{image}=Y.
\]
区别只在于训练阶段监督目标 $Y$ 的定义：
\[
Y=
\begin{cases}
G, & \text{GT LoRA},\\
\bar G, & \text{MTP LoRA},\\
T_{\mathrm{moe}}, & \text{MoE Teacher LoRA}.
\end{cases}
\]
因此，MTP 与 MoE 的复杂处理只发生在训练数据和目标构造阶段；部署时仍然是单个 Qwen2511 LoRA 模型接收原图和指令并输出编辑图。

\subsubsection{阶段一：GT LoRA（基于真值的任务适配）}

\paragraph{目的。}
本阶段回答一个基础问题：在不改变 Qwen2511 主体结构的情况下，仅通过 LoRA~\cite{lora} 微调，是否能够显著提升模型在 MagicBrush 上的任务适配能力。为此，GT LoRA 直接以人工目标图 $G$ 为监督信号，作为后续方法的对照。

\paragraph{Base 表现。}
Qwen2511 Base 不加载任何 LoRA，直接执行 $O=f_{\theta_0}(I,p)$。虽然模型本身具备较强通用编辑先验，但没有针对 MagicBrush 的监督分布进行适配，因此 Target--Output SSIM 仅为 0.450，TO-PSNR 为 14.29，实际表现为能理解多数指令，却容易改错位置或影响背景。

\paragraph{算法与流程。}
GT LoRA 的目标是将 Qwen2511 适配到 MagicBrush 的监督分布。具体做法是冻结 Qwen2511 原始 DiT 权重，只在 attention 与 MLP 等线性层加入低秩更新：
\[
W'=W_0+\Delta W,\qquad
\Delta W=\frac{\alpha}{r}BA.
\]
其中 $A\in\mathbb{R}^{r\times d}$、$B\in\mathbb{R}^{k\times r}$，训练时只更新 $A,B$。本实验以 MagicBrush 真值图 $G$ 作为监督目标，rank 设为 32，学习率为 $1\times10^{-4}$，训练 1 个 epoch，使用 4 张 A100 并行。

Qwen-Image-Edit-2511 的训练流程可抽象为文本编码器、图像 VAE、扩散 DiT 和调度器。每个训练样本包含 $(I,p,Y,M,w_s)$，其中 $Y$ 为当前实验的监督目标，$w_s$ 为样本权重。流程如下：
\begin{enumerate}
  \item 使用 VAE 分别编码目标图 $Y$ 与条件图 $I$；
  \item 使用文本编码器编码编辑指令 $p$；
  \item 对目标隐变量加入噪声并采样时间步 $t$；
  \item DiT 在文本条件和图像条件下预测流匹配速度；
  \item 只更新 LoRA 参数，训练完成后保存 LoRA 权重并在推理时加载。
\end{enumerate}
具体地，首先用 VAE 编码目标图与条件图：
\[
z_Y=\operatorname{Enc}_{\mathrm{vae}}(Y),\qquad
z_I=\operatorname{Enc}_{\mathrm{vae}}(I),
\]
并由文本编码器得到指令表示：
\[
h_p=\operatorname{Enc}_{\mathrm{text}}(p).
\]
随后对目标隐变量采样时间步 $t$ 与噪声 $\epsilon$：
\[
z_t=a(t)z_Y+b(t)\epsilon.
\]
DiT 接收加噪隐变量、文本条件和图像条件，预测流匹配速度：
\[
\hat u_t=D_{\theta_0+\Delta\theta}(z_t,t,h_p,z_I).
\]
优化时只更新 LoRA 参数 $\Delta\theta$；推理阶段将 LoRA 融入 Qwen2511 DiT 后执行标准反向扩散，并由 VAE 解码得到编辑结果。

\paragraph{训练目标。}
为避免大面积背景主导损失，训练采用 mask-aware 的区域归一化流匹配损失。设逐元素速度场误差为：
\[
E=\left\|v_\theta(z_t,t,I,p)-v^\star\right\|_2^2.
\]
将编辑 mask resize 到隐变量尺度后记为 $\hat M$，背景为 $\hat B=1-\hat M$，则：
\[
\mathcal{L}_{\mathrm{edit}}=\frac{\sum \hat M E}{\sum \hat M+\varepsilon},\qquad
\mathcal{L}_{\mathrm{bg}}=\frac{\sum \hat B E}{\sum \hat B+\varepsilon}.
\]
GT LoRA 最终使用：
\[
\mathcal{L}_{\mathrm{sft}}
=
w_s\cdot
\frac{\lambda_e\mathcal{L}_{\mathrm{edit}}+\lambda_b\mathcal{L}_{\mathrm{bg}}}
{\lambda_e+\lambda_b},
\qquad
\lambda_e=1.5,\ \lambda_b=0.5.
\]

\paragraph{结果与分析。}
GT LoRA 将 TO-SSIM 从 0.450 提升到 0.696，说明参数高效微调能显著提升 Qwen2511 在 MagicBrush 上的任务适配能力。但它仍存在两类问题：一是编辑区域定位不稳定，二是背景保持不足，例如局部替换时可能移动无关物体或改变画面布局。

\subsubsection{阶段二：MTP LoRA（目标编辑与背景保持）}

\paragraph{目的。}
GT LoRA 虽然证明了微调有效，但它直接拟合整张目标图 $G$，容易把目标图中的背景漂移也当作监督信号。MTP 的目的就是重写监督目标，使训练样本本身明确表达“编辑区域向目标图靠近、背景区域向原图靠近”。

\paragraph{核心思想。}
MTP（Masked Target Programming）的核心是把直接拟合整张目标图 $G$，改为拟合一个显式保留背景的 clean target：
\[
\bar G=\tilde M\odot G+(1-\tilde M)\odot I.
\]
其中 $\tilde M$ 为 soft edit mask。编辑区域使用目标图，背景区域使用原图，边界通过膨胀和高斯特征做软化，从而把“改目标区域、保留其余区域”的要求直接写入监督信号。

\paragraph{算法流程。}
MTP 的完整处理可以分为七步：
\begin{enumerate}
  \item \textbf{构造差异图：}计算 source-target 像素差异 $D$，得到初始 diff mask；
  \item \textbf{生成候选 mask：}同时考虑 diff mask、dataset mask 及 dataset mask 的反相；
  \item \textbf{清理候选 mask：}通过移除小区域、填洞、闭运算和膨胀去除噪声；
  \item \textbf{候选评分：}根据区域区分度、差异覆盖率和面积惩罚选择最佳 mask；
  \item \textbf{soft mask 化：}对 hard mask 进行膨胀和高斯特征处理，得到 $\tilde M$；
  \item \textbf{构造 clean target：}由 $\bar G=\tilde M\odot G+(1-\tilde M)\odot I$ 得到训练目标；
  \item \textbf{加入 no-op 样本：}额外加入保持样本，强化“无编辑时不改图”的能力。
\end{enumerate}

\paragraph{Mask 生成与选择。}
Mask 构造阶段同时考虑 source-target diff mask、数据集 mask 及其反相候选：
\[
\mathcal{M}=\{M_{\mathrm{diff}},M_{\mathrm{dataset}},1-M_{\mathrm{dataset}}\}.
\]
其中 diff mask 来自原图与目标图的逐像素差异：
\[
D(i,j)=\frac{1}{3}\sum_c |I(i,j,c)-G(i,j,c)|.
\]
对 $D$ 使用阈值得到初始区域后，再通过移除小连通域、填洞、闭运算与膨胀等形态学操作清理。每个候选 mask 的评分由区域区分度、差异覆盖率和面积惩罚组成：
\[
S_M(M)=
d_{\mathrm{in}}(M)-d_{\mathrm{out}}(M)
+\beta c(M)
-\gamma\max(0,a(M)-a_{\mathrm{broad}})
-\eta\max(0,a_{\mathrm{tiny}}-a(M)).
\]
其中 $a(M)$ 是 mask 面积比例，$d_{\mathrm{in}}$ 与 $d_{\mathrm{out}}$ 分别衡量 mask 内外的差异均值，$c(M)$ 表示差异覆盖率。若指令属于全局编辑，或 mask 面积超过阈值，则使用全图编辑，令 $\tilde M=\mathbf{1}$，避免把风格、光照、季节等任务错误压缩成局部替换。

统计上，8807 个训练样本中包含 267 条 no-op 保持样本与 36 条 global edit；8774 条选择 source-target diff mask，25 条选择 dataset mask，8 条选择反相 dataset mask；平均编辑 mask 面积为 0.0959。这说明大多数样本可由 source-target 差异定位编辑区域，但保留 dataset mask 与反相候选可以提高鲁棒性。

局部编辑不能使用硬边界直接拼接。MTP 先对选中的 hard mask $M^\star$ 膨胀，再进行高斯 feather：
\[
M_d=\operatorname{Dilate}(M^\star,r_d),\qquad
\tilde M=\operatorname{clip}\left(\mathcal{G}_{\sigma}*M_d,0,1\right).
\]
同时构造边界带：
\[
M_{\mathrm{bd}}
=
\operatorname{Dilate}(M^\star,r_{\mathrm{bd}})
-
\operatorname{Erode}(M^\star,r_{\mathrm{bd}}).
\]
为了让模型学会“没有编辑要求时不要改图”，MTP 还加入保持样本：
\[
I+\text{``Do not change the image''}\rightarrow I.
\]
这类样本的 mask 为全 0，样本权重较低，只作为背景保持正则。

\paragraph{训练目标。}
MTP 使用与 GT LoRA 相同的 Qwen2511 LoRA 训练流程，但将监督目标设为 $\bar G$，并加入编辑区、背景区和边界区三项区域损失：
\[
\mathcal{L}_{\mathrm{mtp}}
=
\frac{
\lambda_e\mathcal{L}_{\mathrm{edit}}
+\lambda_b\mathcal{L}_{\mathrm{bg}}
+\lambda_{\mathrm{bd}}\mathcal{L}_{\mathrm{bd}}}
{\lambda_e+\lambda_b+\lambda_{\mathrm{bd}}},
\]
其中边界损失定义为：
\[
\mathcal{L}_{\mathrm{bd}}
=
\frac{\sum \hat M_{\mathrm{bd}}E}{\sum \hat M_{\mathrm{bd}}+\varepsilon}.
\]
最终 PhaseA 设置为 $\lambda_e=4.0$、$\lambda_b=0.3$、$\lambda_{\mathrm{bd}}=0.15$，LoRA rank 为 16，学习率为 $5\times10^{-5}$，训练 1 个 epoch。

\paragraph{结果与分析。}
实验结果显示，MTP LoRA 的 TO-SSIM 达到 0.740，高于 GT LoRA 的 0.696；BG-SSIM 从 0.821 提升到 0.828；Edit Change 从 0.043 提升到 0.172。这说明 MTP 不仅提升了目标相似度，也让模型更愿意在正确区域发生有效编辑，同时减少无关背景变化。

\subsubsection{阶段三：MoE-Fusion Teacher（区域级多专家蒸馏）}

\paragraph{核心思想。}
对于 MagicBrush 这种给出了原始图和目标图的数据集来说，用 mask 作为监督信号是可以的，
但是 mask 也有局限，有时对于风格的修改，比如“把夏天改成秋天”，这种指令，是单独 mask 无法准
确反映的，而且对于同一指令，不同区域可能需要不同的编辑策略，而且结果也不是唯一的，目标图
只能作为一种参考，为此想到，能否构建一个教师，能够交给模型一个修改方向而非仅仅是答案呢？
为此希望在基线的三个模型基础上，借鉴混合专家模型思想~\cite{moe}，发挥每个模型的长处，善于生成正确图像的负责生成正确图像，
善于保持背景的负责背景的维持。MoE-Fusion Teacher 的目标是利用多个专家在不同区域的优势构
造训练期 teacher，作为监督微调的目标。真实专家候选为：
\[
\mathcal{C}=\{C_{\mathrm{pix2pix}},C_{\mathrm{qwen}},C_{\mathrm{editar}}\},
\]
实际实现中还为 Qwen 候选加入 conservative/aggressive 两个虚拟变体，以覆盖不同编辑强度：
\[
C_{\mathrm{qwen\mbox{-}cons}}
=
\operatorname{clip}\left(I+s_c(C_{\mathrm{qwen}}-I),0,1\right),
\qquad s_c=0.65,
\]
\[
C_{\mathrm{qwen\mbox{-}aggr}}
=
\operatorname{clip}\left(I+s_a(C_{\mathrm{qwen}}-I),0,1\right),
\qquad s_a=1.25.
\]
因此实际路由的专家族为：
\[
\mathcal{E}=\{\text{pix2pix},\text{qwen},\text{qwen-cons},\text{qwen-aggr},\text{editar}\}.
\]

\paragraph{完整流程。}
MoE-Fusion Teacher 的训练期构造流程如下：
\begin{enumerate}
  \item \textbf{专家候选生成：}由 Pix2Pix、Qwen-Image-Edit、EditAR 分别生成候选图，并为 Qwen 构造 conservative/aggressive 变体；
  \item \textbf{全局候选评分：}计算编辑区相似、背景保持、方向一致性和边界自然度，用于判断候选质量；
  \item \textbf{编辑区域拆分：}将 mask 分解为多个连通区域，对每个区域重新计算局部分数；
  \item \textbf{区域路由：}根据局部分数与置信度选择硬路由或 soft top-$K$ 融合；
  \item \textbf{背景路由：}局部编辑默认使用 source 保持背景，全局编辑则允许背景候选参与排序；
  \item \textbf{融合与回退：}进行颜色协调化和拉普拉斯融合；若融合结果比最佳单专家更差，则回退到最佳候选；
  \item \textbf{蒸馏：}将最终 $T_{\mathrm{moe}}$ 作为监督目标训练单个 Qwen2511 LoRA。
\end{enumerate}

\paragraph{专家评分。}
MoE 对候选图像分别计算编辑区目标相似、背景保持、编辑方向一致性和边界自然度。编辑区目标相似与背景保持定义为：
\[
E_{\mathrm{edit}}(C)=\operatorname{MSE}(C,G;M)
+\alpha_s\left(1-\operatorname{SSIM}(C,G;M)\right),
\]
\[
E_{\mathrm{bg}}(C)=\operatorname{MSE}(C,I;1-M)
+\alpha_b\left(1-\operatorname{SSIM}(C,I;1-M)\right).
\]
编辑方向一致性比较候选相对 source 的变化方向与 target 相对 source 的变化方向：
\[
D_{\mathrm{dir}}(C)=1-\cos\left((C-I)\odot M,(G-I)\odot M\right).
\]
边界惩罚衡量 mask 边界附近的颜色和梯度跳变：
\[
B(C)=
\rho\|C-I\|_{1,M_{\mathrm{bd}}}
+(1-\rho)\|\nabla C-\nabla I\|_{1,M_{\mathrm{bd}}}.
\]
综合得分越低越好：
\[
S(C)=
\lambda_e E_{\mathrm{edit}}(C)
+\lambda_b E_{\mathrm{bg}}(C)
+\lambda_d D_{\mathrm{dir}}(C)
+\lambda_{\mathrm{bd}}B(C)
+\lambda_f\operatorname{MSE}(C,G).
\]
局部评分会在每个连通区域 $M_k$ 上重新计算，并加入 no-change 惩罚：
\[
P_{\mathrm{no\mbox{-}change}}(C,M_k)
=
\max\left(0,\tau_{\mathrm{chg}}-\operatorname{MSE}(C,I;M_k)\right),
\]
以避免几乎不改图的候选因为像素误差较小而被误选。

\paragraph{区域置信度。}
当最优专家和第二名专家的分数分别为 $S_1,S_2$ 时，区域置信度定义为：
\[
\operatorname{conf}
=
\operatorname{clip}\left(
0.55+0.45\tanh
\frac{S_2-S_1}{0.25\max(|S_1|,|S_2|)+\varepsilon},
0.55,1.0
\right).
\]
置信度越高，说明最优专家明显优于其他专家，可以使用硬路由；置信度越低，说明多个专家质量接近，需要使用更柔和的融合策略。

\paragraph{区域级路由与融合。}
不同于整图选择，MoE 将编辑 mask 拆成连通区域：
\[
M=\bigcup_{k=1}^{K}M_k.
\]
每个区域独立选择专家：
\[
e_k^\star=\arg\min_{e\in\mathcal{E}}S_k(C_e).
\]
当最优专家明显优于其他专家时使用硬路由；当多个专家质量接近时，在 top-$K$ 专家之间进行软融合：
\[
\pi_{k,e}
=
\frac{\exp(-S_k(C_e)/\tau)}
{\sum_{e'\in\operatorname{TopK}(k)}\exp(-S_k(C_{e'})/\tau)},
\qquad
C_k^{\mathrm{route}}=\sum_{e\in\operatorname{TopK}(k)}\pi_{k,e}C_e.
\]
背景区域默认选择 source 或背景保持得分最好的专家：
\[
e_{\mathrm{bg}}^\star
=
\arg\min_{e\in\mathcal{E}\cup\{\text{source}\}}E_{\mathrm{bg}}(C_e).
\]
初始融合图为：
\[
T_0
=
(1-\tilde M)\odot C_{e_{\mathrm{bg}}^\star}
+\sum_{k=1}^{K}\tilde M_k\odot C_k^{\mathrm{route}}.
\]
随后通过颜色协调化与拉普拉斯融合降低边界突兀感，最终 teacher 记为
\[
T_{\mathrm{moe}}=\operatorname{Blend}_{\mathrm{lap}}(\operatorname{Harmonize}(T_0,I,M)).
\]
若融合 teacher 的综合得分明显差于最佳单专家，则回退到最佳候选，避免错误融合污染训练目标。

\paragraph{蒸馏到单一 LoRA。}
MoE teacher 只用于训练阶段。学生模型仍是 Qwen2511 source-only LoRA，样本权重由 teacher confidence 调节：
\[
w_s=\operatorname{clip}(w_{\min}+\operatorname{conf}(T_{\mathrm{moe}})(w_{\max}-w_{\min}),w_{\min},w_{\max}).
\]
训练时采用 mask-aware 流匹配损失：
\[
\mathcal{L}_{\mathrm{moe}}=w_s
\frac{\lambda_e\mathcal{L}_{\mathrm{edit}}^{\mathrm{moe}}
+\lambda_b\mathcal{L}_{\mathrm{bg}}^{\mathrm{moe}}}
{\lambda_e+\lambda_b}.
\]

\paragraph{结果与分析。}
在 MagicBrush-dev 上，MoE Teacher LoRA 达到最高 TO-SSIM 0.763、TO-PSNR 19.01 与 BG-SSIM 0.852，说明区域级 teacher 能提供比单一真值或单一 clean target 更强的训练信号。不过，由于 Pix2Pix 和 EditAR 质量显著弱于 Qwen，融合时仍可能引入轻微重影或局部伪影。

\subsubsection{小结}

整体来看，Qwen2511 Base 具有较强通用能力，但在 MagicBrush 的目标一致性和背景保持上不足；GT LoRA 证明了任务微调的有效性；MTP 通过 clean target 将局部编辑与背景保持显式编码进监督目标；MoE-Fusion Teacher 进一步利用多专家候选构造区域级 teacher，并把训练期融合能力蒸馏到单个 LoRA 模型中。指标上，TO-SSIM 由 0.450 依次提升至 0.696、0.740 和 0.763，BG-SSIM 最高提升至 0.852，说明改进方向与“改对且少动背景”的任务目标一致。

后续工作可从三个方向继续推进：其一，引入 GroundingDINO~\cite{groundingdino}/SAM~\cite{sam} 或 VLM planner 构造更可靠的语义 mask；其二，使用更自然的局部 refinement 或扩散式 inpainting 替代简单融合；其三，将 MoE 产生的区域 winner/loser 信息转化为 preference loss，使学生模型学习区域级专家选择依据。

\subsection{InstructPix2Pix 推理优化占位}

InstructPix2Pix 的主要问题在于单次采样不稳定，且 MagicBrush 中的编辑指令往往较短，容易导致模型保守编辑或对目标区域修改不足。另一方面，提高 text CFG 可以增强编辑强度，但也更容易破坏背景。因此，本节将 InstructPix2Pix 的改进定位为推理侧优化：不重新训练模型，而是通过 prompt 改写、多候选搜索、局部裁剪和背景融合提高有效编辑概率。

\paragraph{Prompt + Oracle。}
首先对原始编辑指令进行增强，使其更明确地描述目标变化与保持要求；随后在多组 text CFG、image CFG 和 seed 下生成候选结果。对候选图像进行 oracle 重排时，同时考虑全图相似度与 mask 区域相似度：
\[
S_{\mathrm{oracle}}
=0.4\operatorname{SSIM}_{\mathrm{full}}
+0.6\operatorname{SSIM}_{\mathrm{mask}}.
\]
该策略的核心是把单次不稳定采样转化为多候选选择问题。具体 prompt 改写模板、CFG 搜索范围和候选数量保留占位：\placeholder{补充 prompt 模板和候选搜索配置。}

\paragraph{Mask-Crop。}
对于小目标或边界较清晰的局部编辑，直接在整图上生成会使目标区域占比过小，从而导致编辑不足。Mask-Crop 根据编辑 mask 计算 bounding box，并按一定比例向外扩展后裁出局部区域，再在 crop 上执行编辑。完成后将局部结果缩放回原图位置。该策略适合对象新增、属性修改、局部替换等任务，不适合大范围背景替换或全局风格变化。

\paragraph{BG-Fusion。}
为了减少候选图对非编辑区域的影响，最终结果可以使用 soft mask 将候选图与原图融合：
\[
O_{\mathrm{final}}
=\tilde M\odot O_{\mathrm{cand}}+(1-\tilde M)\odot I.
\]
其中 $\tilde M$ 为经过 feather 的软 mask。该方法可以显著降低背景漂移，但依赖 mask 质量；如果 mask 边界不准确，可能产生边界拼接痕迹或误保留目标区域。定量结果和典型案例保留占位：\placeholder{补充 IP2P 优化前后对比表和案例图。}

\subsection{图层分解与分层编辑占位}

图层分解的目标是显式隔离待编辑对象，使后续编辑尽量限制在目标语义层内。与直接对整图执行编辑相比，分层编辑更适合对象级替换、局部属性变化和前景主体修改等任务。

\paragraph{图层分解与定位。}
输入原图 $I$ 和编辑指令 $p$ 后，首先调用 Qwen-Image-Layered 或类似分层模型，将图像拆解为若干语义图层：
\[
\{L_1,L_2,\ldots,L_K\}=\operatorname{LayerDecompose}(I).
\]
随后使用 CLIP 或其他语义匹配模型，根据编辑指令自动选择最相关图层：
\[
k^\star=\arg\max_k\operatorname{sim}(\phi(L_k),\psi(p)).
\]
其中 $\phi$ 和 $\psi$ 分别表示图像图层与文本指令的编码器。

\paragraph{目标编辑与合成。}
选中目标图层后，仅对该图层或其对应局部区域执行 Qwen-Image-Edit，再通过 alpha composite 与背景画布合成：
\[
\mathrm{dst}_{\mathrm{new}}
=\mathrm{src}\times\alpha+\mathrm{dst}\times(1-\alpha).
\]
该流程能够减少背景和非目标对象的无关变化；但如果分层结果或图层匹配不准，错误会传递到后续编辑阶段。因此，该方法的关键瓶颈在于图层质量、图层数量选择和目标图层识别准确性。具体实现设置与案例图保留占位：\placeholder{补充图层数量、匹配规则和可视化案例。}

\subsection{EditAR 局限与后续优化占位}

EditAR 的改进空间主要来自其自回归视觉生成范式本身。VQ-VAE 重建图像时低频结构通常较稳定，但纹理、高频细节和局部边界与原图存在较大差异；这会使 EditAR 在需要精细保持背景和局部纹理的编辑任务中处于劣势。

此外，EditAR 训练时采用 teacher forcing，而推理时需要根据已经生成的 token 继续预测下一个 token。随着序列长度增加，早期 token 的误差可能逐步累积并影响后续生成，导致局部编辑错位、纹理漂移或目标语义不完整。该问题可概括为：
\[
p(\{y_i\}_{i=1}^{N}\mid I,p)
=\prod_{i=1}^{N}p(y_i\mid y_{<i},\operatorname{tok}(I),\operatorname{tok}(p)),
\]
其中任一前序 token 的偏差都可能影响后续条件分布。

后续优化可以从更高质量的视觉 tokenizer、mask-aware decoding、局部 token 重采样、候选重排和编辑区域优先生成等方向展开。由于本阶段尚未完成系统实验，本节仅保留分析与占位：\placeholder{补充 EditAR 重建误差可视化、token 漂移分析和改进实验。}

\section{Conclusion}

本文围绕指令式图像编辑任务，对 InstructPix2Pix、Qwen-Image-Edit 与 EditAR 三类代表性方法进行了统一复现与对比分析。实验结果表明，三种模型路线在指令执行、背景保持和编辑幅度之间呈现出明显差异：Qwen-Image-Edit 具备最强的语义理解与目标编辑能力，但更容易对背景、风格或非目标区域产生额外改动；InstructPix2Pix 在背景保持上更稳定，但面对复杂语义替换、小目标新增和大范围编辑时存在编辑不足；EditAR 虽然在部分保持类指标上表现不低，但定性结果显示其容易陷入保守失败，并受到 VQ-VAE 重建误差与自回归误差累积的限制。

针对上述现象，本文进一步从训练优化、推理优化和结构化编辑三个方向进行了方法改进。对于 Qwen 系列模型，低步数蒸馏通过学习 4-step student 到 40-step teacher 的残差修正关系，在提升推理效率的同时缓解背景退化与细节漂移；Qwen2511 的 GT LoRA、MTP LoRA 与 MoE-Fusion Teacher LoRA 则分别从任务适配、背景保持和多专家监督信号构造三个层面提升模型表现。对于 InstructPix2Pix，prompt/oracle、多候选搜索、mask-crop 与 BG-fusion 使推理过程更稳定，并增强了局部编辑的可控性。图层分解方法进一步通过显式隔离目标对象减少无关区域改动，而对 EditAR 的分析揭示了自回归视觉编辑在精细保持和长序列生成中的关键瓶颈。

总体而言，本文的复现与改进结果说明，指令式图像编辑的核心并非单纯追求更强的生成能力，而是在“准确执行指令”和“保持无关区域不变”之间建立更可靠的平衡。未来工作可在更复杂的 HQ-Edit~\cite{hqedit}、Reason-Edit~\cite{smartedit} 和 GIE-Bench~\cite{giebench} 等评测集上进一步验证方法泛化性，并结合更精确的语义 mask、区域级偏好学习和高质量视觉 tokenizer，继续提升模型在复杂场景下的稳定性、可控性与编辑质量。
